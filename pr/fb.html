Link Sharer Detector

Reveal who shared the link — extract sharer's Facebook ID / profile identifier

ℹ️ Paste any Facebook post/photo/video URL that was shared by someone. This tool extracts the sharer's Facebook ID (user ID or username) from the share URL pattern. Works with "sharer.php", "sharer/sharer.php", or standard fb link shares.

⚡ How it works: Extracts `u` parameter from `/sharer/sharer.php?u=...` or analyzes URLs that contain `/?u=` pattern. Also tries to fetch sharer info from `fbclid` or share endpoint hints.
💡 Note: For direct post URLs, we attempt to parse the profile ID from path (e.g., /username/posts/... or /profileID/posts/...). The tool focuses on the sharer identity.
