Link Sharer Detector

Reveal who shared the link — extract sharer's Facebook ID / profile identifier

ℹ️ How to get the sharer ID: After someone shares a link, go to their post on Facebook and copy the URL from the browser address bar. It looks like https://www.facebook.com/username/posts/123456789. Paste it here. Do NOT use the "share dialog" link (sharer.php) – it does not contain the sharer's ID.
