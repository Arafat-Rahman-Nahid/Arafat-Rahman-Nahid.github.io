FB Sharer ID Detector

Find the Facebook ID of the person who shared a link (the sharer, not the original poster)

1 Paste the Facebook share link
📌 Example: https://www.facebook.com/share/14TFop7bBsn/
⚠️ The share link itself does NOT contain the sharer's ID. You need to open it while logged in to Facebook.
2 Get the sharer's profile URL
🧭 After opening the share link (step 1) while logged into Facebook:
• Look for the text "Shared by [Name]" near the top of the post.
• Click on the sharer's name to go to their profile.
• Copy the full profile URL from your browser's address bar.
• Paste it below and click "Extract Sharer ID".
✅ Sharer Identification
🆔 Sharer Facebook ID / Profile Identifier
—
⚡ Why this two‑step method? Facebook does not expose the sharer's ID inside the share link itself. The ID is only shown to logged‑in users on the post page. By opening the link and then copying the sharer's profile URL, you can extract the ID with this tool.
💡 Pro tip: If you have many share links, you can bookmark this page and use it repeatedly – just follow the two steps each time.
